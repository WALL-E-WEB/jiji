# globals

定义的变量时，避免no-undef 规则将发出警告

 "globals": {
        "customvar1": "writable", // 允许重写变量
        "customvar2": "readonly" // 只读
     	"Promise": "off" // 关闭全局变量
    }
由于历史原因:
布尔值 false 和字符串值 "readable" 等价于 "readonly"。
布尔值 true 和字符串值 "writeable" 等价于 "writable"。但是，不建议使用旧值。

# rules

"off" or 0 - 关闭规则
"warn" or 1 - 将规则视为一个警告（不会影响退出码）
"error" or 2 - 将规则视为一个错误 (程序会退出)
